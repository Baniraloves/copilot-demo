# GitHub Copilot デモ用 Issue テンプレート

このディレクトリには、GitHub Copilotの機能をデモするためのIssue候補を記載します。

## 📋 デモ用Issue一覧

### 1. Todo編集機能の追加
**優先度**: 高  
**タグ**: `enhancement`, `good first issue`

**説明**:
既存のTodoを編集できる機能を追加してください。

**要件**:
- Todoアイテムをクリックすると編集モードになる
- タイトルと説明を編集できる
- 編集完了後、PUTエンドポイントを使用してバックエンドに送信
- キャンセルボタンで編集を破棄できる

**参考ファイル**:
- `frontend/src/App.tsx`
- `backend/main.py` (PUT エンドポイントは既に実装済み)

---

## 🎯 デモの進め方

1. **Issue作成**: 上記のテンプレートからGitHub Issueを作成
2. **Copilot割当**: IssueにCopilotを割り当て
3. **自動解決**: Copilotが自動でPRを作成
4. **レビュー**: Copilotにレビューを依頼
5. **マージ**: PRをマージして機能追加完了

## 🔧 推奨デモ順序

1. **Todo編集機能** (簡単、視覚的にわかりやすい)
2. **検索・フィルター機能** (UIの拡張、状態管理)
3. **SQLiteデータベース対応** (バックエンドの変更)
4. **E2Eテストの拡張** (Playwrightのデモ)
5. **カテゴリ/タグ機能** (フル機能の追加)
